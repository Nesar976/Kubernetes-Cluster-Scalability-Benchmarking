# Contributing

We welcome contributions to the Kubernetes Cluster Scalability Benchmarking project!

## How to Contribute
1.  Fork the repository.
2.  Create a feature branch.
3.  Submit a Pull Request.

## Guidelines
*   Ensure all scripts are POSIX compliant.
*   Update documentation for any new workloads.
*   Test your changes on a local cluster (e.g., Minikube or index).
